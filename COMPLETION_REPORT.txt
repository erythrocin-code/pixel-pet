🚀 立即开始（推荐方式）
--------------------------------------------------------------------------------

方式一：网页版（最简单！）
1. 找到 web_version.html 文件
2. 用浏览器打开
3. 直接开始玩！
4. 支持手机浏览器

方式二：桌面版（测试用）
1. 安装 Kivy：pip install kivy
2. 运行：python main.py

方式三：Android APK
1. 使用 Google Colab 构建（详见 COLAB_BUILD.md）
2. 或使用 Docker 构建（详见 README.md）
3. 安装到手机
